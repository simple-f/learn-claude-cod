# Learn Claude Code

学习 Claude Code 的示例代码和笔记。

## 目录

- `s01_agent_loop.py` - Agent 循环基础
- `s01_notes.md` - 第 1 课笔记
- `s02_tool_use.py` - 工具使用示例
- `s02_tool_use_annotated.py` - 带注释的工具使用
- `s02_notes.md` - 第 2 课笔记
- `s04_subagent.py` - 子 Agent 示例
- `s09_agent_teams.py` - Agent 团队协作
- `s10_team_protocols.py` - 团队协议示例

## 环境要求

```bash
pip install -r requirements.txt
```

## 运行示例

```bash
python s01_agent_loop.py
python s02_tool_use.py
```
